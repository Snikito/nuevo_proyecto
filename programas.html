Cómo la chipeé
Programas exclusivos de 3ds

Un poco de contexto:

Una consola modificada tiene el poder de instalar cualquier programa (siempre y cuando esté diseñado para la consola). Estos programas son especiales porque ofrecen cosas que la consola base no puede hacer (como instalar juegos truchos).

Programa 1
FBI

El FBI es un programa que toda 3ds chipeada tiene que tener. Con este programa instalamos todos los programas que podamos usar más tarde. Para instalar este programa necesitamos el "Homebrew Launcher".

Programa 2
Homebrew Launcher

Así como el FBI, el Homebrew Launcher instala programas. La diferencia entre este y el FBI es que este nos permite instalar programas con un formato específico (y este formato es muy raro de encontrar). La única aplicación que conozco que use este formato es el instalador del FBI.

Programa 3
Anemone 3DS

Anemone 3DS es un programa que no todos tienen. Esto es porque este programa es solamente para la estética. En Nintendo 3ds tenemos la posibilidad de ponerle un fondo de pantalla a nuestro "escritorio" de la consola. Estos fondos son creados por empresas profecionales, y solo podemos elegir esos. Uno no puede poner la foto que quiere en el "escritorio". Sin embargo, Anemone 3DS nos permite instalar fondos creados por personas comunes, e incluso enseñan cómo hacer tu propio fondo.
